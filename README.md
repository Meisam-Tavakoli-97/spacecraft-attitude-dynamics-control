# Spacecraft Attitude Dynamics and Control

This repository contains the MATLAB and Simulink implementation of a **Spacecraft Attitude Dynamics and Control System (ADCS)** developed as a bachelor-level final project.  
The project focuses on modeling spacecraft attitude dynamics, environmental disturbances, sensor behavior, and control algorithms for detumbling, pointing, and stabilization maneuvers.

> **Author:** M. Tavakoli  
> **Supervisors:** Professor Hamid Moeenfard, Professor Vahid Bohlouri

---

## Project Overview

The implemented ADCS framework includes:
- Spacecraft rigid-body dynamics and kinematics (quaternion-based)
- Environmental disturbance modeling
- Sensor modeling and filtering
- Attitude determination algorithms
- Control allocation using thrusters and magnetorquers
- Detumbling, slew maneuver, and stabilization simulations

All simulations are implemented in **MATLAB/Simulink** and structured to allow modular testing of individual subsystems.

## How to Run the Project

1. **Initialization**
   - Run the main initialization script:
     ```
     BFN_00_main.m
     ```
   This script initializes all required parameters and variables for the simulations.

2. **Full System Model**
   - Open and run:
     ```
     BFN_00_full_system.slx
     ```
   This Simulink model represents the complete spacecraft ADCS architecture.

   The following MATLAB scripts post-process and analyze the results generated by the full system model:
   - `BFN_03_Disturbs.m` ‚Äì disturbance torque analysis
   - `BFN_04_Detumbl.m` ‚Äì detumbling performance (uses `detumbl.mat`)
   - `BFN_05_total_impulse.m` ‚Äì thruster total impulse computation
   - `BFN_06_slew.m` ‚Äì slew maneuver results
   - `BFN_07_stabilization.m` ‚Äì stabilization performance analysis

3. **Gyro Filtering Model**
   - Simulink model:
     ```
     BFN_01_Gyro_filter.slx
     ```
   - Results visualization:
     ```
     BFN_01_Gyro_filter_results.m
     ```

4. **Thruster Model**
   - Simulink model:
     ```
     BFN_02_Thrusters.slx
     ```
   - Results visualization:
     ``
     BFN_02_Thrusters1.m
     ```

## Repository Structure

1. **`BFN_00_main.m`**  
   Main execution script. Must be run first; initializes all variables and parameters.

2. **`BFN_00_full_system.slx`**  
   Simulink model of the complete ADCS system.

   - **`BFN_03_Disturbs.m`** ‚Äì Plots environmental disturbance torques  
   - **`BFN_04_Detumbl.m`** ‚Äì Plots detumbling results (uses `detumbl.mat`)  
   - **`BFN_05_total_impulse.m`** ‚Äì Computes total impulse delivered by thrusters  
   - **`BFN_06_slew.m`** ‚Äì Plots slew maneuver results  
   - **`BFN_07_stabilization.m`** ‚Äì Plots attitude stabilization results  

3. **`BFN_01_Gyro_filter.slx`**  
   Simulink model of the fiber-optic gyroscope and filtering algorithm.

   - **`BFN_01_Gyro_filter_results.m`** ‚Äì Plots gyro filtering results  

4. **`BFN_02_Thrusters.slx`**  
   Thruster model and firing sequence calculation.

   - **`BFN_02_Thrusters1.m`** ‚Äì Plots thruster firing results
